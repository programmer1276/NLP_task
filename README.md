Анализ токсичности комментариев: NLP предобработка и статистический анализ
Проект по обработке естественного языка (NLP). Основная цель — провести полный цикл предобработки текстовых данных и выявить ключевые лексические различия между токсичными и нейтральными комментариями с использованием метрики TF-IDF.

Задачи проекта
Очистка данных: Удаление шума (символов, цифр, спецсимволов).

Нормализация: Лемматизация текста для приведения слов к канонической форме.

Фильтрация: Удаление стоп-слов, не несущих смысловой нагрузки.

Векторизация: Расчет весов TF-IDF для выявления значимых терминов.

Визуализация: Построение распределений топ-10 слов для всего корпуса, а также отдельно для токсичных (1) и нетоксичных (0) комментариев.

Технологический стек
Python 3.11+

Pandas: Манипуляция данными и работа с CSV.

NLTK (Natural Language Toolkit): Лемматизация и работа со стоп-словами.

Scikit-learn: Векторизация текста (TfidfVectorizer).

Matplotlib / Seaborn: Визуализация данных и построение графиков.

Описание решения
Предобработка текста
Для обеспечения высокого качества анализа я реализовал следующий пайплайн:

Regex Cleaning: Использование регулярных выражений для удаления всего, кроме латинских букв. Это исключает влияние пунктуации и случайных чисел на статистику.

Lemmatization: В отличие от стемминга, лемматизация (WordNetLemmatizer) учитывает контекст и корректно преобразует слова (например, feet -> foot), что сохраняет семантическую точность.

Stop-words filtering: Использование расширенного списка стоп-слов NLTK позволяет сфокусироваться на уникальных характеристиках текста.

Векторизация (TF-IDF)
Вместо обычного частотного анализа (Bag of Words) был выбран метод TF-IDF (Term Frequency - Inverse Document Frequency).

Это позволяет математически снизить вес слов, которые часто встречаются во всем корпусе (например, "wikipedia", "article"), и выделить слова, специфичные для конкретных категорий (особенно для класса toxic).

Использована L2-нормализация, чтобы исключить влияние длины комментария на его итоговый вес.

Результаты
Программа генерирует графики распределения весов слов.

Общий анализ: Показывает наиболее обсуждаемые темы в датасете.

Сравнительный анализ: Наглядно демонстрирует, как меняется лексическое ядро при переходе от нейтрального общения к токсичному, что подтверждает корректность разметки данных.
